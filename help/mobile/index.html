Turnarchist

Mobile Controls

Turnarchist is turn-based: monsters only act when you do. On mobile, most actions are done with swipes and long-press.

Controls:

Move / attack: Swipe up/down/left/right

Keep moving: Swipe and hold (movement repeats while your finger stays down)

Actions (attack / pick up / interact / examine): Long-press on a target to open the action menu

Inventory: Tap the Inventory button (bottom-right)

Use / equip: Tap an item in your quickbar/inventory

Rearrange items: Touch + move to start dragging, then release on a slot

Drop item: Drag an item outside the inventory UI, then release

Full map: Tap the minimap to open/close the full map

Skills: Tap the XP counter to open the skills menu

Bestiary: Tap the Bestiary button to open/close

Menu: Tap the Menu button to open/close

Tip: swipes that start on UI (inventory, menus, etc) are treated as UI gestures, not movement. Swipe on the game world to move.
